\# Namnlista



Detta är en enkel Streamlit-app för att pricka av närvarande studenter vid ett event.



\## Funktioner

\- Ange eventnamn

\- Klistra in studentnamn

\- Kryssa i närvarande studenter

\- Se sammanställning direkt

\- Ladda ner resultatet som CSV



\## Filer

\- `namnlista\_streamlit.py`

\- `requirements.txt`



\## Kör lokalt

Installera först beroenden:



```bash

pip install -r requirements.txt
